Master Mobile UX

Mobile first allows for much easier transition to desktop

Designing mobile first is much more cost effective.

Users can reach buttons most easily at the bottom of the screen. Try to keep buttons at the bottom of the screen rather than the top.

Use svg rather than png or jpg. This allows the connection with the server not to be slowed down.
